# Frontend TODO

## 主工作流

当前主链路：

```text
粗糙想法 / PRD 草稿 / 外部资料
  -> 创意工坊 AI skill 分析
  -> 候选气泡与用户确认补充
  -> 写入灵感气泡画布
  -> 画布整理、追问、归类、快照
  -> PRD 输出中心按标签/分类分束
  -> 可编辑章节草稿
  -> Markdown / PDF 导出
```
